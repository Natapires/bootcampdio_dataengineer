# 📊 Fundamentos de Engenharia de Dados

> Repositório de estudos e projetos desenvolvidos durante o bootcamp de **Fundamentos de Engenharia de Dados e Machine Learning**, cobrindo desde os conceitos essenciais de Python e banco de dados até Machine Learning e Cloud Computing com AWS.

---

## 🎯 Sobre o Bootcamp

Este curso oferece uma base sólida em Python e banco de dados, fundamentos de ETL e Cloud, e em como evoluir com Machine Learning aplicando cada conceito em projetos que simulan desafio reais do mercado de dados, com uma trilha que já inclui IA generativa e integração de LLMs em pipelines.

---

## 📚 Módulos do Curso

### 🐍 1. Princípios de Python e Versionamento de Código com Git e GitHub


---

### 🗃️ 2. Introdução à Ciência de Dados e Python
Fundamentos de modelagem e manipulação de dados em bancos relacionais. Aborda:
- Modelagem entidade-relacionamento (ER)
- Linguagem SQL: `SELECT`, `JOIN`, `GROUP BY`, subqueries e mais
- Boas práticas de normalização
- Integração entre banco de dados e Python

---

### 🔄 3. Introdução a Banco de Dados Relacionais e ETL
Processos e técnicas para movimentação e transformação de dados entre sistemas. Aborda:
- Conceitos e arquitetura de pipelines ETL
- Extração de dados de múltiplas fontes (arquivos, APIs, bancos)
- Transformações e padronizações de dados
- Carregamento em destinos estruturados (Data Warehouse, banco de dados)

---

### 📈 4. Análise de Dados com Excel e Copilot
Uso do Excel como ferramenta analítica potencializada por Inteligência Artificial. Aborda:
- Tabelas dinâmicas, gráficos e dashboards
- Fórmulas avançadas e análise estatística descritiva
- Uso do **Microsoft Copilot** para geração automática de insights
- Interpretação de dados e storytelling com dados

---

### ☁️ 5. Noções de Cloud com AWS
Introdução ao ambiente de nuvem da Amazon Web Services voltado para dados. Aborda:
- Conceitos fundamentais de computação em nuvem (IaaS, PaaS, SaaS)
- Principais serviços AWS para dados: `S3`, `RDS`, `Glue`, `Athena`
- Armazenamento e processamento de dados na nuvem
- Segurança, escalabilidade e boas práticas na AWS

---

### 🤖 6. Fundamentos e Técnicas de Machine Learning
Introdução ao aprendizado de máquina com foco em aplicação prática. Aborda:
- Conceitos de aprendizado supervisionado e não supervisionado
- Algoritmos clássicos: Regressão Linear, Árvore de Decisão, K-Means e outros
- Treinamento, validação e avaliação de modelos
- Bibliotecas `Scikit-learn` e `Matplotlib` para modelagem e visualização

## 🚀 Como Usar Este Repositório

1. Clone o repositório:
```bash

```

2. Acesse a pasta do módulo desejado:
