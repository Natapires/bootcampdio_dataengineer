# 📊 Fundamentos de Engenharia de Dados e Machine Learning

> Repositório de estudos e projetos desenvolvidos durante o bootcamp de **Fundamentos de Engenharia de Dados e Machine Learning** da [DIO](https://www.dio.me/), cobrindo desde os conceitos essenciais de Python e versionamento de código até Machine Learning, Cloud com AWS e IA Generativa aplicada a pipelines de dados.

---

## 🎯 Sobre o Bootcamp

Este bootcamp oferece uma base sólida e progressiva para quem deseja atuar na área de dados. A trilha começa com os fundamentos de Python e banco de dados, avança por ETL e Cloud, e culmina em técnicas de Machine Learning — com projetos práticos que simulam desafios reais do mercado. O curso também inclui conteúdos de ponta como **IA Generativa** e **integração de LLMs em pipelines de dados**.

---

## 📚 Módulos do Bootcamp

### 🐍 1. Princípios de Python e Versionamento de Código com Git e GitHub

Primeiro contato com a linguagem Python e com as ferramentas de controle de versão mais usadas no mercado.

- Fundamentos da linguagem Python
- Versionamento de código com **Git e GitHub**

---

### 🔬 2. Introdução à Ciência de Dados com Python

Mergulho profundo nas estruturas de dados e recursos da linguagem Python, base indispensável para qualquer cientista ou engenheiro de dados.

- Tipos de operadores
- Estruturas condicionais e de repetição
- Manipulação de strings
- Listas, Tuplas, Conjuntos e Dicionários
- Funções em Python
- Prática com operadores e estruturas condicionais

---

### 🗃️ 3. Introdução a Banco de Dados Relacionais e ETL

Fundamentos de bancos de dados e do processo de ETL, com um diferencial moderno: uso de **IA Generativa** dentro dos pipelines.

- Banco de Dados Relacionais com **SQL**
- Banco de Dados **NoSQL**
- Fundamentos de **ETL** (Extract, Transform, Load) com Python
- IA Generativa aplicada a pipelines de ETL com Python
- Prática de ETL com Python

---

### 📈 4. Análise de Dados com Excel e Copilot

Uso do Excel como poderosa ferramenta analítica, potencializada pelo **Microsoft Copilot** com Inteligência Artificial.

- Introdução ao **Excel 365**
- Trabalhando com **Microsoft Copilot**
- Usando Copilot para escrever consultas **SQL**
- Processo de ETL com **Excel e Power Query**
- Análise de bases de dados com Excel e Copilot
- Introdução a Dashboards
- 🏆 Projeto: **Dashboard de Vendas do Xbox com Excel**

---

### ☁️ 5. Noções de Cloud com AWS

Introdução ao ecossistema da **Amazon Web Services**, explorando os principais serviços voltados para computação, redes, armazenamento e banco de dados na nuvem.

- Como a computação em nuvem funciona
- Infraestrutura da AWS
- Computação na AWS
- Redes na AWS
- Armazenamento e banco de dados na AWS
- 🏆 Projeto: **Redução de Custos em Farmácias com AWS**

---

### 🤖 6. Fundamentos e Técnicas de Machine Learning

Formação completa em Machine Learning, abrangendo desde os algoritmos clássicos até redes neurais, métodos bioinspirados e aplicações práticas com Python.

- Introdução ao Machine Learning
- Métodos Bioinspirados
- Redes Neurais Artificiais
- Algoritmos Genéticos
- **SVM** (Support Vector Machine)
- Classificação de problemas e exploração de datasets
- Linguagens de programação para ML
- 🏆 Projeto: **Python para Machine Learning na Prática**

## 🚀 Como Usar Este Repositório

1. Clone o repositório:
```bash
git clone https://github.com/Natapires/bootcampdio_dataengineer
```

2. Acesse a pasta do módulo desejado:
```bash
cd 01-Principios_Python
```

3. Instale as dependências (quando necessário):
```bash
pip install -r requirements.txt
```
